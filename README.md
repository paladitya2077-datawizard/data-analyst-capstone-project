# Data Analyst Capstone Project: Restaurant Marketing Analytics

## 📊 Project Overview 
* **Goal:** To analyze restaurant data to identify top-performing locations and restaurants and help improve customer recommendations.
* **Dataset:** This capstone project builds a matrix to identify star restaurants and generate recommendations across 9,000+ rows of restaurant business data.

## 🛠️ Tech Stack
The following tools have been used to build the dashboard
* **Python (Pandas, NumPy, Matplotlib, Seaborn):** Used for Data inspection, manipulation and cleaning.
* **Tableau Public:** Used for Data visualization and dashboard building.
---

## ⚙️ Project Workflow (How it Was Made)
1. **Data Prep:** Cleaned 9,000+ rows of data, removed duplicates, and handled null values.
2. **Feature Engineering:** Standardized different types of currency (Rs, IDR, R$, etc.) into one single currency (USD) for accurate avg cost analysis.
3. **Exploratory Analysis:** Used Seaborn heatmaps and bar charts to identify correlations between restaurant services and ratings.
4. **BI Delivery:** Exported the cleaned and combined dataset into Tableau Public to create the final business matrix and generate restaurant recommendations

---

## 💡 Key Insights (What I Found)
* **Insight 1:** Restaurants offering online delivery and table booking services get significantly higher ratings and customer votes.
* **Insight 2:** Customers generally prefer low-to-mid-range priced restaurants; higher prices do not equal to better ratings.
* **Insight 3:** New Delhi, India is the primary market with over 5,400 Restaurants active.
* **Insight 4:** North Indian cuisine is the most dominated and served cuisine across all restaurants globally across the dataset

# Data Analyst Capstone Project: Marketing Analytics

## Project Overview & Problem Statement
A restaurant consolidator was looking to revamp the B2C portal using intelligent automation tech.
This capstone project builds a matrix to identify star restaurants and generate recommendations.

## The Hybrid Data Architecture
This project implements an end-to-end analytics workflow across two major platforms:
1. **Python (Data Wrangling & EDA):** Handling duplicate and null values, merging tables, outlier filtering, feature engineering.
2. **Tableau (Executive Business Intelligence):** Using the cleaned data from Python to build interactive, live-filtered performance dashboards.

---

## Tech Stack & Dependencies
- **Language:** Python 3.14.5
- **Libraries Used:** Pandas, NumPy, Matplotlib, Seaborn
- **BI Platform:** Tableau Public / Tableau Desktop

---

## Step-by-Step Methodology

### 1. Data Processing & Exploratory Data Analysis (Python)
- Performed preliminary data inspection and reported the findings as the structure of the data, missing values, duplicates, etc. on (9,000+ rows  and 19 columns).
- Removed duplicates and fixed null values.
- Merged the tables and used feature engineering to make a new column.
- Wrote code to convert different types of currency(Rs, IDR, R$, etc.)  into one single currency(USD).
- Used Seaborn and Matplotlib to plot barcharts, histograms, correlation heatmap to identify features driving key business metrics.
- *The Jupyter Notebook containing the full code can be reviewed in the `/notebooks` directory.*

### 2. Dashboard Engineering (Tableau)
- Imported the cleaned dataset generated by the Python workflow.
- Established a clean visualization layout focusing on restuarants overview, scatterplot and matrix to identify the star restaurant.
- Built active dashboard actions (interactive filtering, hover tooltips) to allow stakeholders to drill down by ratings, votes and avg cost.

---

## Core Business Insights & Visual Delivery

### Key Takeaways:
- **Insight 1:** Restuarants offering features and options such as Table Booking and Online delivery because such Restaurannts are likely to get more ratings and votes from the customers.
- **Insight 2:** Customers generally prefer low to mid range restaurants often giving them higher votes and rating indicating that expensive does not instantly equal to high ratings and votes.
- **Insight 3:** New Delhi, India is the primary market with over 5,400 Restaurants active.
- **Insight 4:** Analysis explains that North Indian cuisine is the most dominated and served cuisine across all restaurants globally.
